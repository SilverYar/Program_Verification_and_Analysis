Лаба 2.


Основные программы:

   - yaroslav.smv (C:\istarToNusmv\NuSMV\bin)      - исходная модель со всеми спецификациями
   - yaroslav_s1.smv (C:\istarToNusmv\NuSMV\bin)   - исходная модель с нарушенной первой спецификацией
   - yaroslav_s2.smv (C:\istarToNusmv\NuSMV\bin)   - исходная модель с нарушенной второй спецификацией
   - yaroslav_s3s4.smv (C:\istarToNusmv\NuSMV\bin) - исходной модели с нарушенной третьей и четвертой спецификациями

1. Перейти в 
C:\istarToNusmv\NuSMV\bin
в проводнике

2. Открыть cmd 

	cd C:\istarToNusmv\NuSMV\bin	

	
3. Ввести команду
	1) NuSMV yaroslav.smv  // запуск исходной модели со всеми спецификациями
результат показать В.М. Ицыксону
	2) NuSMV yaroslav_s1.smv  // запуск исходной модели с нарушенной первой спецификацией
результат показать В.М. Ицыксону
	3) NuSMV yaroslav_s2.smv  // запуск исходной модели с нарушенной второй спецификацией
результат показать В.М. Ицыксону
	4) NuSMV yaroslav_s3s4.smv  // запуск исходной модели с нарушенной третьей и четвертой спецификациями
результат показать В.М. Ицыксону
